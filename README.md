## Clean-Up & Simple Run

### Install Pods (iOS Only)

- `npm i`
- `cd ios && pod install`
- `cd .. && flutter run`

### Android local.properties (Android Only)

- `npm i`
- `cd android && mkdir local.properties`
- `nano local.properties`
